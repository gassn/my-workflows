`todo-cli` の README.md を作成しました。TypeScript 製 CLI としての基本的な体裁を整え、以下の要素を盛り込んでいます。

## 構成

- **概要**: ツールの目的を 1 文で説明
- **特徴**: 機能の箇条書き（追加・一覧・完了マーク）
- **必要要件**: Node.js 20 以上を明記し、バージョン確認コマンドも記載
- **インストール**: npm グローバルインストールとソースからの手順の両方をカバー
- **使い方**: `add` / `list` / `done` の各コマンドの実行例と出力例
- **コマンド一覧**: 表形式で一覧化（`--help` / `--version` も含む）
- **データの保存場所**: `~/.todo-cli/tasks.json` を想定
- **開発**: `npm install` / `npm run dev` / `npm run build` / `npm test`
- **ライセンス**: MIT

## ポイント

- 日本語ベースで、コマンドラインの例は英語出力で現実味を持たせました
