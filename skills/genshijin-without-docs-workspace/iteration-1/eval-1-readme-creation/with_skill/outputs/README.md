# todo-cli

`todo-cli` は、TypeScript で実装されたシンプルなコマンドラインタスク管理ツールです。ターミナルから素早くタスクを追加し、一覧を表示し、完了マークを付けることができます。

## 特長

- タスクの追加
- タスクの一覧表示
- タスクの完了マーク付け
- TypeScript による型安全な実装
- ローカルファイルへのタスク永続化

## 必要環境

- Node.js 20 以上
- npm 10 以上 (Node.js 20 に同梱)

## インストール

グローバルインストールする場合は、以下のコマンドを実行してください。

```bash
npm install -g todo-cli
```

リポジトリからソースコードを取得してローカルで使う場合は、以下の手順で進めてください。

```bash
git clone https://github.com/example/todo-cli.git
cd todo-cli
npm install
npm run build
npm link
```

## 使い方

### タスクの追加

`add` サブコマンドに、タスクの内容を文字列として渡します。

```bash
todo-cli add "牛乳を買う"
```

実行すると、新しいタスクが内部ストレージに追加され、割り当てられた ID が表示されます。

### タスクの一覧表示

登録されているすべてのタスクを確認するには、`list` サブコマンドを使用します。

```bash
todo-cli list
```

出力例は次のとおりです。完了済みタスクにはチェックマークが付与されます。

```
[ ] 1. 牛乳を買う
[x] 2. 本を返却する
[ ] 3. レポートを書く
```

### タスクの完了マーク付け

タスクを完了済みにする場合は、`done` サブコマンドに対象のタスク ID を指定してください。

```bash
todo-cli done 1
```

指定した ID のタスクが完了状態に更新されます。

## コマンド一覧

| コマンド | 説明 |
| --- | --- |
| `todo-cli add <text>` | 新しいタスクを追加します |
| `todo-cli list` | 登録済みのタスクを一覧表示します |
| `todo-cli done <id>` | 指定した ID のタスクを完了済みに変更します |

## データ保存場所

タスクはユーザーのホームディレクトリ配下にある `~/.todo-cli/tasks.json` に JSON 形式で保存されます。ファイルを直接編集することも可能ですが、フォーマットを壊さないよう注意してください。

## 開発

ローカルで開発する場合は、以下のスクリプトを利用できます。

```bash
npm run build    # TypeScript を JavaScript にコンパイルします
npm run dev      # ソースコードを監視しながらビルドします
npm test         # ユニットテストを実行します
npm run lint     # コードスタイルをチェックします
```
